## For Windows Users (Easiest Method)

1. **Double-click** `SETUP_WINDOWS.bat`
   - This will install everything automatically
   - Wait for it to finish (1-2 minutes)

2. **Double-click** `START_DASHBOARD.bat`
   - Dashboard opens in your browser automatically!

3. **Done!** ðŸŽ‰

---

## For Mac/Linux Users

1. Open Terminal in this folder

2. Run setup:
   ```bash
   ./setup.sh
   ```

3. Run dashboard:
   ```bash
   ./start.sh
   ```

4. **Done!** ðŸŽ‰

---

## Manual Method (All Operating Systems)

If the scripts don't work, do it manually:

```bash
# 1. Install dependencies
npm install

# 2. Start dashboard
npm run dev
```

Dashboard opens at: **http://localhost:3000**

---

## First Time Setup Only

- You only need to run setup ONCE
- After that, just run `START_DASHBOARD.bat` (Windows) or `./start.sh` (Mac/Linux)

---

## Stopping the Dashboard

- Press `Ctrl + C` in the terminal
- Or just close the terminal window
